Bienvenido al Taller 2

A continuación, exploraremos los conceptos fundamentales del diseño web moderno: el Modelo de Caja (Box Model), la flexibilidad de Flexbox y el poder estructural de CSS Grid Layout.

Modelo de Caja (Box Model)

El Modelo de Caja es un concepto fundamental en CSS que describe cómo se estructuran los elementos HTML. Cada elemento se considera como una caja compuesta por contenido, relleno, borde y margen.
