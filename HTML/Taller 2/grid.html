Modelo de Grid (CSS Grid Layout)

El Modelo de Grid es un sistema de diseño bidimensional que permite a los desarrolladores crear layouts complejos de manera sencilla y eficiente. Con CSS Grid, puedes definir filas y columnas, y colocar los elementos en cualquier posición dentro de esa cuadrícula, utilizando las propiedades de bordes(border), rellleno(padding) y el contenido(content)
